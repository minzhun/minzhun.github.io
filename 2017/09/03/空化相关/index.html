空化相关

Sep 3, 2017 | 专题

1 空化现象

  • 液体内部或液体-固体界面处，蒸气或气体空泡形成、发展、溃灭的过程
    • 液体内部存在空化核
    • 局部压力低于液体饱和蒸气压力
  • 空泡溃灭时产生的冲击力，可对物体造成损伤

2 空化模型

  • CFD 中，通常放置在多相流模型之下，并被视作相变问题之一
  • 表示液体蒸发速率与凝结速率，即，气泡的产生和消失
  • 作为源项，出现在液体体积分数输运方程的右端

- Rayleigh-Plesset Equation

  • 无限流体中的球状气泡控制方程
    • $R$：气泡半径
    • $p_{B}$、$p_{\infty}$：气泡内部压力、无限远处压力
    • $\rho_{l}$、$\nu_{l}$：液体密度、液体运动粘度
    • $\sigma$：气泡表面张力

- 常出现的空化模型

  • 基于Rayleigh-Plesset Equation 的空化模型
    • 下面三种模型，在不考虑 Rayleigh-Plesset Equation 中的二阶项与表面张力项的基础上得到，推导过程类似，在质量净产生速率的表达式上有所区别
    • Singhal Model
    • Zwart-Gerber-Belamri Model
    • Schnerr-Sauer Model
  • 基于经验得出的比例化空化模型
    • Kunz Model：气泡产生、消失速率的表达形式不同
    • “比例化空化模型”的说法在若干文献中有出现，但确切定义未找到

- CFD 软件中的空化模型

  • Fluent
    • Singhal Model
    • Zwart-Gerber-Belamri Model
    • Schnerr and Sauer Model
  • CFX
    • The Rayleigh Plesset Model
  • Star-CCM+
    • Full Rayleigh–Plesset Model
    • Schnerr-Sauer Model

3 空化模型 @ADINA-CFD

  • 主相（Primary Species）设置为气体，第一相（First Species）设置为液体
  • 液体体积分数的输运方程
  • Kunz Model
    • $p_{v}$：液体饱和蒸气压力
    • $t_{\infty}$、$v_{\infty}$：速度尺度、时间尺度
    • $C_{prod}$、$C_{dest}$：液体蒸发系数、冷凝系数，需参考同类问题或通过试验校准
  • Zwart-Gerber-Belamri Model（ZGB model）
    • $p_{v}$：液体饱和蒸气压力
    • $f_{nuc}$：空化核体积分数，推荐取0.0005
    • $R$：气泡半径，推荐取1e-6
    • $C_{prod}$、$C_{dest}$：液体蒸发系数、冷凝系数，推荐取50、0.01
  • 对湍流计算的影响
    • 通过对混合物密度进行修正，体现气体压缩性对湍流粘度计算的影响
    • $n$：大于等于1；随空化数降低、雷诺数升高而升高
  • 一些使用限制
    • 与 VOF 功能一起使用
    • 只适用于FCBI-C 单元
    • 只适用于瞬态分析
    • 不适用于可压缩流动与自由表面流动

4 Notes

  • 空化现象仿真，不只是选择空化模型，需要考虑的还包括：
  • 湍流模型
    • 选择合适的模型
    • 边界条件与初始条件
    • 壁面网格
  • 多相流模型
    • 选择合适的模型
    • 满足模型要求的网格与时间步长
  • 空化模型
    • 选择合适的模型
    • 确定模型中必要的参数
  • 旋转机械（如果）
    • 动参考系（Moving Reference Frame）
    • 多参考系（Multiple Reference Frame）
    • 滑移网格
    • 周期边界
  • 方程求解
    • 速度与压力：分离求解 or 耦合求解
    • 时间步长
    • 松弛系数
