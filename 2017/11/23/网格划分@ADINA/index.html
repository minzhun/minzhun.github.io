网格划分 @ADINA

Nov 23, 2017 | 软件与实践

1 网格划分的一些准备

  • 几何处理
    • 去除某些特征、抽中面等
  • 确定单元
    • 单元类型、单元形状、单元大小
  • 几何分块
    • 将几何切分成拓扑结构简单的若干部分

2 几何类型 @ADINA

  • AUI Native Geometry
    • Point、Line、Surface、Volume
  • ADINA-M
    • 几何引擎（可选）：Parasolid 或 OpenCASCADE
    • Point、Edge、Face、Body

3 网格尺寸控制

方式 1：Subdivision

  • 设置长度
    • 均分
  • 设置份数
    • 均分、非均分

方式 2：Point-Size

  • 网格尺寸根据Point-Size，在各个点之间逐渐变化
    • 首先，设置尺寸控制方式为 Use End-Point Sizes
    • 之后，根据需要，按，体、面、线、点的顺序设置 Point-Size

Other

  • Automatic Grading
    • 设置部分区域的网格尺寸，自动向外设置其他区域的网格尺寸
    • 适用：Body / Face、Free-Form、Tetra / Tri
  • Curvature-based Sizing
    • 根据曲率设置网格尺寸
    • 适用：Body / Face、Free-Form、Tetra / Tri、Delaunay
    • 参数：GEO-ERROR（越小越密）、SAMPLING（越大越密）
  • Size Function
    • 可以在设置Point-Size 时使用
    • 可以在 Body / Face 的 Tetra / Tri 的自由网格划分时使用
  • Max-Size
    • 可使内部网格尺寸大于边界网格尺寸
    • 适用：Body / Face、Free-Form、Tetra / Tri、Delaunay

4 映射网格划分（Mapped Meshing）

  • 适用
    • 二维：拓扑为四边形、三角形
    • 三维：拓扑为六面体、三棱柱
  • 规则网格
    • 对边网格划分份数相等
  • 不规则网格
    • 对边网格划分份数不相等
    • 会存在一些三角形过渡
  • Pattern
    • 网格划分的形式，例如，三角形网格斜边的方向
    • Tri：Pattern 1 - Pattern 9
    • Quad：Pattern 10 - Pattern 11
  • Degenerate
    • 三角形区域、三棱柱形区域
    • 四边形、六面体是否退化为三角形、棱柱体
  • Mid-Node
    • 高阶单元，边，中节点布置
    • Straight、Project、Curved
  • Body Sweep / Body Revolve
    • 将 Face 上的二维网格，拉伸/旋转为三维网格（六面体、棱柱体）
    • Face 上的网格，可以为映射网格划分，也可以为自由网格划分

5 自由网格划分（Free-Form Meshing）

三角形/四面体

  • 方法
    • Advancing Front
    • Delaunay
  • Advancing Front
    • Growth Rate 不起作用
  • Delaunay
    • Growth Rate：控制网格的过渡，1.0（慢） - 3.0（快）
    • Optimize for：Meshing Speed（划分更快速）、Subdivision Accuracy（过渡更平缓）
    • Curvature-based Sizing：按曲率控制大小时，Optimize for 不起作用

四边形/六面体

  • 要求全四边形网格时
    • 所有边界网格划分份数之和为偶数
  • Pyramid
    • 对于三维自由网格划分，一般都会存在过渡形状，例如，四面体、金字塔
    • No（全部使用四面体过渡）
    • Yes（六面体和四面体之间，使用金字塔过渡）
    • Boundary Only（边界上使用金字塔，其他使用四面体，不使用六面体）

Other

  • Mid-Node
    • 高阶单元，边，中节点布置
    • Straight、Project（推荐）、Curved
  • Mid-Face-Node
    • 高阶单元，面，中节点布置
    • Tri（对角线中点位置）、Quad（面形心位置）
  • Nlayer
    • 设置较窄区域网格层数
    • 适用：Body、Free-Form、Tetra
    • 较窄区域网格划分
  • Boundary Layer
    • 边界层网格，三维时形状可选六面体或棱柱体
    • 适用：Body / Face

6 节点重合及其控制

  • Face-Link
    • 使网格在相邻的几何之间连续
  • Nodal Coincidence
    • 网格划分时，检查并控制节点是否重合
  • 节点的合并/分离操作
    • Join、Detach、Split

7 其他

  • 单元形状与节点数
    • 三角形：3节点、6节点、7节点
    • 四边形：4节点、8节点、9节点
    • 四面体：4节点、10节点、11节点
    • 六面体：8节点、20节点、27节点
    • 四边形（壳单元）：4节点、8节点、9节点、16节点
    • 棱柱体、金字塔：六面体的退化形状
    • 三角形（壳单元）：四边形（壳单元）的退化
