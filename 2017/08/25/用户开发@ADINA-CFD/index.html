用户开发 @ADINA-CFD

Aug 25, 2017 | 软件与实践

1 用户定义子程序功能及名称

- 定义边界条件

  • 【ADFUBC】User-Supplied Boundary Conditions
  • 【DCLUBC】User-Supplied Boundary Conditions for FCBI-C Elements

- 定义与修改材料

  • 【ADFUSR】User-Supplied Material Data for Incompressible Flows without Turbulent-KE or Turbulent-KW Models
  • 【CFVUSR】User-Supplied Material Data for Compressible Fluid Flows
  • 【USRMAT】User-Supplied Material Data for Incompressible, Slightly Compressible and Low Speed Compressible Flows
  • 【ADFURM】User-Supplied Material Data and Source Terms for Mass Transfer
  • 【ADFUCM】User-Supplied Material Data and Source Terms for Mass Transfer for FCBI-C Elements

- 定义电荷密度与源电流密度

  • 【ESUSRS】User-Supplied Static Electric Source

- 修改控制参数

  • 【ADFUCP】User-Controlled Parameters

2 用户开发的具体实现

- Step1 安装 Intel Fortran 编译器

  • 不是必须安装 Visual Studio
  • 编译器版本要求见 Question QG-007

- Step2 修改文件 ../usrdll/ifort14.bat

  • 在文件中指定 Intel Fortran 编译器的路径

- Step3 编写用户定义子程序

  • 在 ../usrdll/ovlf0u.f 文件中修改相应子程序
  • 子程序名称及功能，见本篇第1部分

- Step4 将用户定义子程序编译为动态链接库

  • 打开 Windows 命令提示符界面
  • 进入 ../usrdll 文件夹
  • 启动编译环境：ifort14
  • 执行编译：nmake /f Makefile.afusr

- Step5 开始使用

  • ADINA-CFD 的用户定义子程序使用 Heading 作为标识
  • 分析中的 Heading 需要与子程序中的相对应
