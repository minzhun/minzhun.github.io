流固耦合分析方法

Jul 18, 2017 | CAE之术：方法

1 多物理场耦合现象

  • 下面所列出的两场耦合可以互相组合，形成多物理场耦合现象

- 结构-流体

  • 流体：流体载荷作用在结构上
  • 结构：结构变形改变流场边界
  • 分析域不重合，在界面耦合

- 结构-热

  • 热场：使结构变形；影响结构材料本构
  • 结构：塑性变形生热；接触摩擦生热；热传导影响温度分布
  • 分析域可重合，在分析域重合部分或界面耦合

- 结构-电

  • 电场：电场使压电材料发生变形
  • 结构：变形使压电材料发生极化，产生电场
  • 通过本构关系耦合

- 流体-热

  • 固体（不发生变形）与流体之间发生传热现象
  • 固体传热形式：传导
  • 流体传热形式：传导、对流、辐射
  • 在界面耦合

- 流体-电磁

  • 电磁场：产生的洛伦兹力可驱动流体
  • 分析域可重合，在分析域重合部分以体积力的形式耦合

- 热-电

  • 电流：产生焦耳热
  • 分析域可重合，在分析域重合部分以热源的形式耦合

2 流固耦合 （Fluid Structure Interaction，FSI）分析方法

  • FSI 使用的耦合分析方法，也适用于具有类似耦合特点的其他耦合分析

- 单向耦合与双向耦合

  • 单向耦合：结构在流场中受力，但结构的位移或变形对流场的影响可以忽略
  • 双向耦合：结构的位移或变形对流场的影响不能忽略

- 直接耦合（强耦合）

  • 直接求解流场与结构的耦合控制方程
  • 耦合条件（力、位移）在控制方程及其系数矩阵层面得到满足
  • 适用于耦合程度非常高的流固耦合问题，这些问题往往只能通过直接耦合方法求解，例如，
    • 流场中的柔性结构
    • 边界可变的封闭区域中的不可压缩流体
  • 所需内存较大，对于大规模问题可能无法接受
  • 求解过程中，矩阵出现病态的可能性较大
  • 可实现直接耦合的软件
    • ADINA、INTESIM

- 迭代耦合（弱耦合）

  • 分别求解流场与结构的控制方程
  • 耦合条件通过界面处流场与结构之间力、位移的传递与迭代实现
  • 适用于耦合程度不高的问题，在迭代耦合与直接耦合均能够求解时，二者应能给出相同的答案
  • 所需内存相对较少，对于大规模问题更加适合，前提是迭代耦合能够求解此类问题
  • 可实现迭代耦合的软件
    • 单一平台：ADINA、INTESIM、ANSYS Workbench
    • 接口软件：MpCCI

- FSI 中的网格

  • FSI 中，除耦合条件的实现与控制方程的求解外，流场的网格变形问题是另一个难点
  • 目前的处理手段
    • 动网格技术：ALE Formulation，网格重叠法等
    • 自适应网格重划
