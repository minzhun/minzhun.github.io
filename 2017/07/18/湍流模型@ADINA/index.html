湍流模型 @ADINA

Jul 18, 2017 | CAE之技：实现

1 ADINA中的湍流模型的布局

  • 湍流模型数量不多，但基本满足工程需求
  • 目前没有提供雷诺应力模型（RSM）
  • ADINA 的湍流模型在材料管理中进行选择

- 不可压缩流动、微可压缩流动、低速可压缩流动

  • Flow Model：Laminar
    • Large Eddy Simulation Model
  • Flow Model：Turbulent K-Epsilon
    • Standard
    • RNG
    • Realizable
    • 2-Layer Model：可结合上面三种模型使用
  • Flow Model：Turbulent K-Omega/SST
    • High Reynolds
    • Low Reynolds
    • SST：可选择是否开启
  • Flow Model：Turbulent Spalart-Allmarsa
    • Spalart-Allmars Model
    • Detached Eddy Simulation Model：可选择是否开启

- 高速可压缩流动

  • Flow Model：Turbulent K-Epsilon
    • Standard

2 Large Eddy Simulation Model

  • 形式上，LES 属于零方程湍流模型，故放置在层流材料中
  • 入口条件
    • LES 模型可在速度入口附加随机扰动
  • $ \mu_{0} $：流体材料自身的粘度
  • $ \mu_{t} $：湍流粘度

- Subgrid-Scale Model： Standard

  • 输入参数：$ k_{D} $ （无量纲系数）
  • 壁面第一层网格 y+
    • ≤ 1 ？

- Subgrid-Scale Model： Smagorinsky-Lilly

  • 输入参数：$ k_{D} $ （无量纲系数），$ \kappa $（冯卡门系数，默认0.41）
  • 壁面第一层网格 y+
    • 30~200

- Subgrid-Scale Model： RNG Based

  • 输入参数：$ C_{RNG} $（无量纲系数，默认0.09348），$ k_{D} $ （无量纲系数，考虑热传导时需要输入）
  • 只用于FCBI-C 单元

3 K-Epsilon Model

  • $ \mu_{t} $：湍流粘度
  • 入口条件
  • 初始条件
    • 一般与入口条件相同
  • 第一层网格 y+
    • 未使用 2-Layer Model：30~200
    • 使用 2-Layer Model：<1
  • 受壁面函数影响较大

4 K-Omega Model

  • $ \mu_{t} $：湍流粘度
  • 入口条件
  • 初始条件
    • 一般与入口条件相同
  • 第一层网格 y+
    • 5~200
  • 与K-Epsilon Model 相比，数值计算上更加稳定

5 Spalart-Allmarsa Model

  • $ \mu_{t} $：湍流粘度
  • 注意：$ \tilde \nu $ 在ADINA 前后处理中用K 表示
  • 入口条件
  • 初始条件
    • 一般与入口条件相同
  • DES：主要为解决高Re 数出现较大区域分离的流动问题，在ADINA 中：
    • 网格精细区域：SA 模型起SGS 模型的作用
    • 其他区域：SA 模型起RANS 模型的作用
    • 实现方法：用 $ \tilde d$ 代替SA 模型中的 $d$（距壁面的距离），$C_{DES}$ 默认0.65
    • 网格大小 $ \Delta $：起一个开关作用

6 壁面函数

  • 0 < y+ < 200 范围内，对壁面律能够较好地进行拟合

7 近壁面网格

  • 形状要求：正交性好，与壁面平行布置若干层后，过渡到Free-Form Mesh
  • 大小要求：满足湍流模型对y+ 的要求

8 入口条件

  • 若有实验数据，应予以使用
  • $ i $：湍流强度（0.01~0.1）
  • $ v $：速度尺度
  • $ l $：长度尺度
    • 内部流动：水力直径
    • 外部流动：边界层厚度的40%左右

9 说明

  • 根据所分析的问题选择合适的湍流模型，因为目前还没有一个普遍适用的湍流模型
  • 除非有试验或其他手段校准，一般不修改湍流模型内部参数
