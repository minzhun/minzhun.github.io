守恒定律及其描述

Jul 28, 2017 | CAE之道：原理

运动的描述

  • 构型（Formulation）：表示物体所占据的空间，用物体所包含的粒子的位置描述

- 拉格朗日描述

  • 参考构型为初始构型
  • 材料坐标：粒子在初始构型中的坐标
  • 用材料坐标和时间描述物理性质
  • 多用于固体力学
  • FEA 中的网格是运动的
  • 着眼历程

- 欧拉描述

  • 参考构型为当前构型
  • 空间坐标：粒子在当前构型中的坐标
  • 用空间坐标和时间描述物理性质
  • 多用于流体力学
  • CFD 中的网格一般是固定不动的
  • 着眼当下，即此时此刻

物质导数与雷诺输运定理

  • 以微分形式和积分形式体现了拉格朗日描述与欧拉描述之间的联系
  • $ f $：张量
  • $ \mathrm{v} $：速度矢量

- 物质导数

  • 一般形式
  • 考虑质量守恒

- 雷诺输运定理

  • 一般形式
  • 考虑质量守恒
